**Projeto1 - Classificação binária** 

**Dataset**: German credit risk - Kaggle Link: https://www.kaggle.com/uciml/german-credit

**Objetivo**: prever o risco de crédito de um cliente, utilizando um conjunto de dados históricos contendo algumas características dos clientes (variáveis preditoras) e se o cliente é bom ou mau pagador (variável alvo).

**Variáveis**: 9 variáveis preditoras + 1 variável alvo (Risk = good/bad) 

**Métricas**: Acurácia, f1-score e ROC  

**Tratamento nos dados**: tratamento de valores missing/outliers, normalização e balanceamento de classes.

**Algoritmos de aprendizagem** : algoritmo de classificação Random Forest 

**Construção**: dez/21
