**Projeto1** 

**Dataset**: House Prices - Advanced Regression Techniques - Kaggle Link: https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data 

**Objetivo**: prever o preço de venda de um imóvel, utilizando um conjunto de dados históricos contendo algumas características dos imóveis (variáveis preditoras) e o valor pelo qual ele foi vendido (variável alvo), no passado.

**Variáveis**: 75 variáveis preditoras + 1 variável alvo (SalePrice = valor de venda em USD) 

Obs: a descrição de cada variável está em anexo no jupyter notebook. 

**Métricas**: R², MAPE e RMSE 

**Tratamento nos dados**: tratamento de valores missing, outliers, padronização e seleção recursiva de variáveis (RFE).

**Algoritmos de aprendizagem** : Regressão linear multivariada (modelo base) e Regressão linear regularizada ELASTIC NET (modelo final).
